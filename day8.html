Version control

What is it?

  • Tracks changes to files
  • Saves work in multiple locations
  • We'll use git and GitHub
  • GitHub is a site that hosts code for lots of people
  • If you're reading this, you're looking at something hosted on GitHub!
  • Explore GitHub projects

Account setup

Sign up for an acoount at GitHub using your school email. Make sure you know your password

Repository

  • Folder containing all your files
  • Includes a log of all your changes
  • Can access old versions easily

Setup (web-based)

  • Go to GitHub
  • Click the green "New Repository" button
  • Name your repository tech-ready-python
  • Check the box for "Initialize this repository with a README"
  • Choose the "Python" .gitignore
  • Click the green "Create Repository" button

Uploading Files (web-based)

  • Go to your repository
  • Click the grey "Upload files" button
  • Upload the files you need to run the game (including images!)
  • Click the green "Commit changes" button
  • You can also edit files from the web interface - select the file and click the pencil icon to edit
