Functions

  • Reusable chunks of code
  • Some are pre-defined
  • We can define our own functions

Simple function


							def my_simple_function():

								return

def


							def my_simple_function():
								print("I'm not a very useful function")
								return

def is a keyword that tells python we're defining a function

name


							def my_simple_function():
								print("I'm not a very useful function")
								return

my_simple_function is the name of our function. It's how we access it

parameters


							def my_simple_function():
								print("I'm not a very useful function")
								return

() indicates that this is a function. In a more advanced function, we would include parameters here

colon


							def my_simple_function():
								print("I'm not a very useful function")
								return

: indicates that this is the end of this line and that we'll be creating a block of code after it that will run when this line is executed

block of code


							def my_simple_function():
								print("I'm not a very useful function")
								return

The indented code is what will run when the function is called (used)

return


							def my_simple_function():
								print("I'm not a very useful function")
								return

return is a keyword that will exit the function. In a more advanced function, it can make the function return a value just like the input() function

Return values

Some functions return a value, like input()


							def doubled(num):
							return num * 2
							
							print(doubled(5))

The code above will print 10. When doubled(5) is called, it returns 10, so the line is equivalent to print(10)

Lists

  • Ordered list of data
  • Comma separated values in brackets
  • Can be any type of data
  • Only one data type per list
  • Start counting at 0

Accessing values in a list


						my_list = ["alphabet", "book", "cat", "dog", "elephant"]

						print(my_list[1])
						print(my_list[2])
						print(my_list[-1])
						print(my_list[0:3])

The number in square brackets refers to the index of an item

Adding values to a list


						my_list = ["alphabet", "book", "cat", "dog", "elephant"]

						print(my_list)
						my_list.append("frog")
						print(my_list)

Checking if an item is in a list


						my_list = ["alphabet", "book", "cat", "dog", "elephant", "frog"]
						
						print("alphabet" in my_list)
						print("chocolate" in my_list)
						
						if "dog" in my_list:
						print("We've already included dogs in the list")
						else:
						my_list.append("dog")

This works on strings too

Looping


							my_list = ["alphabet", "book", "cat", "dog", "elephant", "frog"]
							
							for item in my_list:
							print(item)

This works on strings too

Checking length


						my_list = ["alphabet", "book", "cat", "dog", "elephant", "frog"]

						print(len(my_list))

						while len(my_list) < 10:
							my_list.append(input("Add another item to the list: "))
						
						print(len(my_list))

This works on strings too
