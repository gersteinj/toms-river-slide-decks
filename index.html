Running Code

  • Can run a program saved in a text file
  • Can use REPL/interactive shell

REPL

  • Read Evaluate Print Loop
  • Quickly run simple code
  • Run a line of code, and display the return value
  • Test code and see what it does

Return Values

  • Each line returns a certain value when evaluated
  • Try entering numbers. What's returned?
  • Try text. Make sure it's in quotes.

Printing

  • Printing is different from returning values
  • print("Put something in here to print it") will display the text in quotes in the console
  • What is returned?
  • print() is a function

Math

  • Python can do math
  • Mathematical operators: + , - , * , / , ** , %
  • The math library
  • Data types: integer & float

Comparisons

  • Comparison operators are same as in math class
  • Try some comparisons. Note any errors you get
  • Data types: boolean

Strings

  • Text
  • Always in quotes

Variables

  • Name representing a value
  • Store values to use again later
  • Create or modify with assignment statements
  • x = 5

User input

  • input() is a function, much like print
  • Action: prompts user for input
  • Returns: user input as a string
  • Optional: Create a prompt using a string in the parentheses
  • Try it in REPL - can you save the value to use again later?

Formatting

  • Many built in ways to format strings
  • Built in tools for many common uses
  • "ToMs RiVeR".title()
  • "Hello from Toms River"[0:5]

Inserting Values

  • "Hello {}, how are you today?".format("Fred")
  • Can have multiple values - separate with commas
