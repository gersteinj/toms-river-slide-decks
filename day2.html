Conditional Statements

How they work

  • if some condition is true, then do something
  • if some condition is true, then do something. Otherwise do some other thing
  • if it's raining out then take an umbrella with you.
  • if the user is logged in, then show their profile. Otherwise show the login screen

Coding it


							my_num = 10

							if my_num == 5:
								print("You got it!")
